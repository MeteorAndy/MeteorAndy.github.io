英语兔语法笔记

英语兔语法笔记

语法总体体系

句子

所有长句子的组成是由简单句（把所有句子拆到不能再拆）构成

  • 绝大多数句子就是“什么+怎么样”，即“主语（人/物）+谓语（广义上的动作/发生了什么事）”

基本句型

  1. 无需承受者（宾语）：主语 + 不及物动词
  2. 一个宾语，无补充信息：主语 + 单及物动词 + 宾语
  3. 两个宾语：主语 + 双及物动词 + 间接宾语 + 直接宾语
  4. 一个宾语 + 补充信息：主语 + 复杂及物动词 + 宾语 + （宾语）补语
  5. 赋予信息：主语 + 系动词 + 主语补语/表语

严格意义上来说，句子中除了主语的余下部分全都是谓语，谓语动词只是谓语当中的一部分内容

句子成分

  1. 主语

  2. 谓语动词

    1. 可以独立完成的动作：不及物动词
    2. 有一个动作的承受者：单及物动词
    3. 有两个动作承受者：双及物动词
    4. 只有一个动作承受者：复杂及物动词
    5. 非“动作”：系动词

  3. 宾语

  4. 宾语补语

  5. 主语补语

  6. 定语（主要用来修饰主语或者宾语）

    例：The little white rabbit ate a large carrot.

    其中，“The little white”是“rabbit”的定语，“a large”是“carrot”的定语

  7. 状语（主要用来修饰谓语动词）

    例：The rabbit ate quickly.

    其中，“quickly”就是“ate”的状语

  8. 同位语（主要用来把主语或宾语再说一遍）

    例：Papa Rabbit, an English teacher, eats carrots.

    其中，“an English teacher”是“Papa Rabbit”的同位语

插入语，如：“This carrot, I think（插入语）, is very tasty.”但是插入语是独立的，不作句子成分。

复合句/并列句：把简单句简单地叠加，是并列关系。

复杂句/复合句：一个简单句套另一个简单句，是从属关系，有着主句和从句的区别。从句就是把一个简单句修改一下来充当另一个句子的成分（谓语动词除外）。如：

  1. 把一个简单句修改一下来充当另一个句子的主语，这就是主语从句
  2. 把一个简单句修改一下来充当另一个句子的宾语，这就是宾语从句
  3. 把一个简单句修改一下来充当另一个句子的表语，这就是表语从句
  4. 把一个简单句修改一下来充当另一个句子的同位语，这就是同位语从句

以上的都称之为名词性从句

  1. 把一个简单句修改一下来充当另一个句子的定语，这就是定语从句
  2. 把一个简单句修改一下来充当另一个句子的状语，这就是状语从句

词性（词类）

  1. 动作（动词）——五种基本句型

    1. 可以独立完成的动作

      主语 + 不及物动词

      例：Papa Rabbit sleeps.

      其中，sleeps是不及物动词

    2. 有1个动作的承受者

      主语 + （单）及物动词 + 宾语

      例：Papa Rabbit likes you.

      其中，likes是（单）及物动词，you是宾语

    3. 有2个动作承受者

      主语 + 双及物动词 + 间接宾语 + 直接宾语

      例：Papa Rabbit teaches you English

      其中，teaches是双及物动词，you是间接宾语，English是直接宾语

    4. 只有1个动作承受者（但需补充）

      主语 + 复杂及物动词 + 宾语 + （宾语）补语

      例：Papa Rabbit considers you smart.

      其中，considers是复杂及物动词，you是宾语，smart是（宾语）补语

    5. 非“动作”

      主语 + （连）系动词 + （主语）补语/表语

    五个基本句型，本质上就讲动词，以及动词需要几个对象才能完整表达意思

  2. 名词（表人和物）

    例：Papa Rabbit is a rabbit.

  3. 冠词（说明人和物）

    例：Papa Rabbit is a rabbit.

  4. 代词（替代人和物）

    例：I am a rabbit.

  5. 形容词（形容人和物）

    例：I am a smart rabbit.

  6. 数词（表数量）

    例：I ate two carrots.

  7. 副词（修饰动作或者形容词）

    例：I ate two carrots quickly.

  8. 介词（表示和其他词关系的词）

    例：I ate two carrots with chopsticks.

  9. 叹词（表感叹）

    例：Ah, the carrot is tasty!

  10. 连词（连接词和句）

    例：I ate two carrots and a potato, because I was hungry.

谓语动词的“三大本领”

  1. 表示动作的时间

    1. 现在
    2. 过去
    3. 将来
    4. 过去将来

  2. 表示动作的状态

    1. 一般
    2. 完成
    3. 进行
    4. 完成进行

    以上两者合在一起称之为时态

一般状态 完成状态 进行状态 完成进行状态
现在时间 一般现在 现在完成 现在进行 现在完成进行
过去时间 一般过去 过去完成 过去进行 过去完成进行
将来时间 一般将来 将来完成 将来进行 将来完成进行
过去将来时间 一般过去将来 过去将来完成 过去将来进行 过去将来完成进行

  1. 表示动作的假设、情感…（语气）

    如虚拟语气、陈述语气、祈使语气

    英语的谓语动词本身往往是不能够独立完成这些本领，以及另外一些其他的本领，而且谓语动词独自也无法完成表示否定、可能性、必须性等意思

助动词（辅助谓语动词表示更多含义的词，以eat举例）

  1. （协助）表示“吃过了（完成）” ==》 have ==》 have eaten

  2. （协助）表示“正在吃（进行）” ==》 be ==》 is eating

  3. （协助）表示“被吃” ==》 be ==》 is eaten

  4. （协助）表示“有能力吃” ==》 can ==》 can eat

  5. （协助）表示“有可能吃” ==》 might ==》 might eat

  6. （协助）表示“有义务吃” ==》 must ==》 must eat

  7. （协助）表示“不吃（否定）” ==》 do ==》 do not eat

千万不要把助动词和它其他的身份（作实义动词时）弄混淆

非谓语动词

动词除了充当句子中的谓语动词之外还有可能充当：

  1. 主语 （将谓语动词修改为非谓语动词）

  2. 宾语 （将谓语动词修改为非谓语动词）

  3. 宾语补语 （将谓语动词修改为非谓语动词）

  4. 主语补语 （将谓语动词修改为非谓语动词）

  5. 定语 （将谓语动词修改为非谓语动词）

  6. …

非谓语动词几乎可以取代所有的从句，从而简化句子。只不过这样的动词就不再具有表示动作时间、状态、语态、语气的功能

是以不同的形式出现在句子中，如：

  1. 动词不定式

    不定式和谓语动词原形一样，如无上下文，则无法判断

  2. 现在分词

  3. 动名词

    现在分词和动名词与谓语动词的进行状态一样，如无上下文，则无法判断

  4. 过去分词

    过去分词和谓语动词的完成时态一样，如无上下文，则无法判断

英语语法的核心就是动词，动词能够串起几乎所有语法概念
